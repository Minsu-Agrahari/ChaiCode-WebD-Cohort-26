## вљ™пёЏ Complete Setup Guide

To run and test the TypeScript files locally, follow the setup instructions below. Since Node.js primarily understands JavaScript natively, we compile the TypeScript files (`.ts`) into JavaScript (`.js`) using the TypeScript Compiler (`tsc`).

1. **Install Dependencies:**
   Make sure you have Node.js installed, then install your node modules (which should include TypeScript if configured in package.json). Run:
   ```bash
   npm install
   ```

2. **TypeScript Compilation (Auto-compile):**
   To continuously watch for any changes in your `.ts` files and compile them into the `dist/` directory, open a terminal in the root of this app and run:
   ```bash
   tsc --watch
   ```
   *(This ensures that whenever you save a `src/*.ts` file, the corresponding `.js` file is immediately generated/updated in `dist/`.)*

3. **Running the Compiled Code:**
   Open a separate terminal to run the compiled JavaScript file via Node.js smoothly. You can use `--watch` to automatically re-run whenever compilation updates the file. For example:
   ```bash
   node --watch ./dist/April-2026/substring_word.js
   ```
