# 01-Weekend-Classes

This directory contains learning materials, practice codes, and the structured curriculum for the core weekend learning sessions.

## 📁 Folder Structure

```text
01-Weekend-Classes/
├── 01-Tools/                        # Git and version control fundamentals
├── 02-Fundamentals/                 # Programming and web fundamentals
├── 03-Building Blocks/              # HTML & CSS essentials
├── 04-Interacting with Browser/     # JavaScript and DOM manipulation
└── 05-Backend Engineering/          # Node.js, Express, and backend concepts
```

## 🎯 Overview
- **Tools**: Learn version control workflows using Git/GitHub.
- **Building Blocks**: Dive deep into layout and styling using HTML and CSS.
- **Interacting with Browser**: Handle DOM manipulation and user events using modern JavaScript.
- **Backend Engineering**: Build robust servers using Node.js and frameworks.
