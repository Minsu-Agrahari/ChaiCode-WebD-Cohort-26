# Cursor Landing Page

This is my first complete frontend website project. I built this landing page to practice basic HTML and CSS concepts and to understand how real websites are structured.

## What I Learned

While building this project, I learned many important frontend concepts:

* How to use an **iframe** to add one HTML file inside another.
* The basics of **Flexbox** for layout.
* Different Flexbox alignment properties like:

  * `space-around`
  * `space-evenly`
  * `space-between`
* How to **center elements using CSS Grid**.
* How to structure a full landing page from start to finish.

## Mistakes I Made

Since this was my first complete frontend project, I made many mistakes:

* Improper layout alignment.
* Confusion between Flexbox and Grid.
* Rewriting sections multiple times.
* Not organizing code properly at first.

## What I Gained from This Project

Even with the mistakes, this project helped me:

* Understand real-world layout techniques.
* Learn how Flexbox and Grid actually work in practice.
* Gain confidence in building a full website from scratch.
* Improve my problem-solving skills.

## Conclusion

This project was an important milestone in my learning journey. It was my first full frontend website, and even though I made many mistakes, I learned a lot from the process.
